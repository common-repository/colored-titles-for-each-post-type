== Description ==
This plugin is build to to choose custom colors for each post , custom post type and page. Apply the choosen color on frontend accordingly.


[Twitter](https://twitter.com/fmjtech/)

== Installation ==

= Installation =
1. Download the zip file and upload the contents to your plugins directory (defaul `wp-content/plugins`)
2. Activate the plugin through the 'plugins' page in WP.
3. Click on Custom Title Colors in left "Admin Menu" and set colors for each post type and page.
